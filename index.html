Example 1:

Associar elemento com uma ação

Click on link below to run action

Action 1 
	<script>
		events.context('example-1',{
			action1:{			
				elements:'#element1_ex1',
				callback:function(arg){
					alert('action 1');
				}
			}
		});
	</script>

Using a element HTML

<a onclick="events.context('example-6').action6('6');">Action 6

OR

Execute the code:

events.context('example-6').action6('6');

API:

Attritube on TAG:

data-events="context:action:event"

context = "id_of_context"
action = "function_of_action"
events = "click,mousemove,keyup,..."
